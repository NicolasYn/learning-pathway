Règles de Learning Pathway

Cette page résume les différentes règles que comporte Learning Pathway.

Pour vous :

  • Learning Pathway n'est pas un site servant à faire sa pub pour son blog, son site personnel, etc...

  • Si vous n'approuvez pas certains sites en lien, ou que certaines pages n'ont pour vous aucune utilités, vous pouvez nous contacter via l'adresse mail learning-pathways at protonmail.com

Un lieu d'apprentissage collaboratif :

  • Respectez le travail des autres collaborateurs : tout le monde à le droit d'apprendre, de modifier ou de créer.
  • Si un contributeur enlève ce que vous avez modifier c'est que cette personne avait une bonne intention de le faire et ne remodifier pas par la suite.
  • Vérifiez que le contenu que vous voulez insérer n'existe pas déjà pour éviter la duplication de contenu.
  • Si vous faites des erreurs ne vous inquiètez pas, d'autres contributeurs le remarqueront et les corrigeront. Toutes les modifications sont conservées dans l'historique, il est donc impossible de perdre des informations.

Merci à tous les contributeurs, le site se construit grâce à vous.
